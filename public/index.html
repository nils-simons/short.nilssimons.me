# CREATE A URL

To create a url just POST following payload to short.nilssimons.me/create:

{
    "url": "https://{some long url}"
}

# GET A URL

To get a url infos just POST to short.nilssimons.me/{yourhash}


# USE A URL

To use a url just GET short.nilssimons.me/{yourhash}
